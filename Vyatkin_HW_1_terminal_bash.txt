1. Посмотреть где я - pwd
2. Создать папку - mkdir folders
3. Зайти в папку - cd folders
4. создать три папки - mkdir folder_1 folder_2 folder_3
5. Зайти в любую папку - cd folder_1
6. Создать 5 файлов (3 txt, 2 json) - touch file_{1,2,3}.txt file_4.json file_5.json
7. Создать 3 папки - mkdir f{1,2,3}
8. Вывести список содержимого папки - ls la
9. Открыть любой txt файл - vim file_1.txt
10. Написать туда что-нибудь, любой текст - Нажать клавишу "А" или "I" английскую, ввести текст - Hello!
11. сохранить и выйти - нажать клавишу "Esk", потом нажать "shift + :", потом клавиши "W","Q" должно получиться ":wq"
12. Выйти из папки на уровень выше - cd ..
13. Переместить любые два файла, которые создали, в любую другую папку - mv ./folder_1/{file_1.txt,file_4.json} ./
14. Скопировать любые 2 файла, которые вы создали, в любую другую папку - cp ./folder_1/{file_3.txt, file_5.json} ./folder_2/
15. Найти файл по имени - find . -name "file_5*"
16. Посмотреть содержимое файла в реальном времени - tail -f filname | grep search_term
17. Вывести несколько первых строк из текстового файла - head filename
18. вывести несколько последних строк из текстового файла - tail filename
19. Просмотреть содержимое длинного файла - less filename
20. Вывести дату и время - date
